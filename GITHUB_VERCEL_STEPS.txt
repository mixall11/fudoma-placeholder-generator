═══════════════════════════════════════════════════════════════════════════════

1. Choď na https://github.com/settings/tokens
2. Klikni "Generate new token"
3. Vyber "Personal access token (classic)"
4. Vygeneru sa ti token: ghp_xxxxxxxxxxxxxx
5. Skopíruj si ho (NEPOS, NEPUBLISH!)
6. Keď Git pýta heslo, vlož tento token

KROK 3: VERCEL (5 MINÚT)
═══════════════════════════════════════════════════════════════════════════════

1. Choď na https://vercel.com
2. Klikni "Sign Up"
3. Vyber "Continue with GitHub"
4. Autorizuj Vercel
5. Klikni "Add New" → "Project"
6. Vyber "Import Git Repository"
7. Nájdi svoj repo "fudoma-placeholder-generator"
8. Klikni "Import"
9. Vercel automaticky detekuje nastavenia
10. Klikni "Deploy"
11. ČAKAJ - vidíš progesbar (1-2 minúty)
12. ✅ Deployment successful = HOTOVO!

VÝSLEDOK: https://fudoma-placeholder-generator.vercel.app

"Cannot connect to GitHub"
→ Regeneruj nový Personal Access Token
→ Alebo nastavuj SSH key

"npm: command not found"
→ Nainštaluj Node.js: https://nodejs.org

"Port 3000 je obsadený"
→ npm run dev -- --port 3001
