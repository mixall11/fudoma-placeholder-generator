## ⚡ Najrýchlejšia cesta (TL;DR):

```bash
# 1. Skopíruj projekt
cp -r /home/claude/vercel-project ~/fudoma-placeholder-generator
cd ~/fudoma-placeholder-generator

# 2. Nainštaluj dependencies
npm install

# 3. Testuj lokálne
npm run dev
# Otvoriť http://localhost:3000

# 4. Vytvor GitHub repo a pushni
git init
git config user.name "Tvoje Meno"
git config user.email "tvoj@email.com"
git add .
git commit -m "Initial commit"
git remote add origin https://github.com/USERNAME/fudoma-placeholder-generator.git
git push -u origin main

# 5. Choď na https://vercel.com, importuj repo a deploy
# 👉 Vercel sama všetko zvládne!
```

**Za 15 minút je to LIVE na internete!** 🚀
